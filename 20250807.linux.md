# 🐧 Linux Basic Command Practice

리눅스 기초 명령어 실습 

---

## 📦 실습 목표

- 디렉토리 생성 및 이동
- 파일 생성 및 수정
- 파일 이름 변경 및 출력
- 권한 없는 파일 삭제 연습

---

## 🧪 실습 명령어 기록

```bash
# 1. 디렉토리 생성 및 이동
mkdir linux_mission
cd linux_mission

# 2. 빈 파일 생성
touch today.txt

# 3. 파일에 내용 작성
echo "지금은 2025년 8월 7일. 나는 리눅스를 배우고 있다." > today.txt

# 4. 파일 내용 출력
cat today.txt

# 5. 파일 이름 변경
mv today.txt log.txt

# 6. 현재 디렉토리 파일 목록 상세 보기
ls -l

# 7. 상위 디렉토리로 이동
cd ..

# 8. 디렉토리 삭제
rm -r linux_mission
```
