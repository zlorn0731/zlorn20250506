## 🧱 실습 코드 ①: Student 클래스 실습
```java
public class Student {
    // 1. 필드 정의
    String name;
    int id;
    int score;

    // 2. 메소드 정의
    void introduce(){
        System.out.println("제 이름은 " + name + "입니다.\n제 학번은 " + id + "입니다.");
    }

    void showscore(){
        System.out.println("제 점수는: " + score + "입니다.");
    }

    // 3. 메인 메소드: 실행부
    public static void main(String[] args) {
        // 4. 객체 생성
        Student s1 = new Student();
        s1.name = "박지안";
        s1.id = 20230844;
        s1.score = 90;

        // 5. 메소드 실행
        s1.introduce();
        s1.showscore();
    }
}

🖨️ 실행 결과
제 이름은 박지안입니다.
제 학번은 20230844입니다.
제 점수는: 90입니다.
```

## 🧪 실습 코드 ②: 파라미터 사용 예제
```java
public class Greeting {

    // 파라미터 name을 받아 인사하는 메소드
    void sayHello(String name) {
        System.out.println("안녕하세요, " + name + "님!");
    }

    public static void main(String[] args) {
        Greeting g = new Greeting();

        g.sayHello("지안");
        g.sayHello("지현");
        g.sayHello("태은");
    }
}
```
## 💡 파라미터란?
메소드가 외부에서 값을 전달받기 위해 선언하는 변수

| 용어              | 설명             | 예시                           |
| --------------- | -------------- | ---------------------------- |
| 파라미터(parameter) | 메소드 선언부에 있는 변수 | `void sayHello(String name)` |
| 인자(argument)    | 메소드 호출 시 넘기는 값 | `sayHello("지안")`             |

메소드 안에서 name은 파라미터
호출할 때 "지안"은 인자

### 📝 오늘의 학습 요약
